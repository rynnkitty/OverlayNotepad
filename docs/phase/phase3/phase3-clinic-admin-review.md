## 항목별 검증 결과

### ✅ 통과

1. **시스템 트레이 동작**: 최소화 시 트레이로 숨김, 더블클릭 복원 — 업무 중 화면 정리에 유용한 패턴. 행정 직원이 다른 프로그램(EMR, 청구 SW) 사용 중에도 빠르게 메모장 접근 가능
2. **서식 전체 적용**: 부분 서식 미지원은 올바른 결정. 행정 메모 용도에서는 전체 서식 일괄 적용이 직관적
3. **자동 저장 연동**: Phase 2에서 구축한 자동 저장이 서식 변경 시에도 유지되는 것은 필수. 확인 완료
4. **컨텍스트 메뉴 구성**: 우클릭 한 번으로 주요 설정에 접근 가능한 구조 — 교육 비용 최소화

### ⚠️ 주의

1. **글꼴 선택 UI**: "시스템 설치 글꼴 선택"이 드롭다운 전체 목록이면 수백 개가 나열됨. 최근 사용 글꼴 상단 노출 또는 자주 쓰는 글꼴 필터 권고
2. **색상 선택기 접근성**: 색상 선택기가 너무 복잡하면 빠른 업무에 방해. 프리셋 색상 팔레트 + 커스텀 색상 조합 권고

## 파라미터 조정 권고

| 항목 | 원래값 | 권고값 | 근거 |
|------|--------|--------|------|
| 글꼴 선택 방식 | 전체 목록 | 프리셋 5~7개 + 시스템 글꼴 다이얼로그 | 빈번 사용 시 빠른 선택 필요 |
| 색상 선택 방식 | 자유 색상 선택기 | 프리셋 8~10색 팔레트 + ColorDialog | 반복 작업에서 빠른 선택 우선 |

## 리스크 및 대안

- **리스크**: 이 프로젝트는 의료 EMR이 아닌 범용 투명 메모장이므로 행정 전문 관점의 검토 범위가 제한적
- **참고**: 업무 보조 도구로 사용 시 "항상 위" + "투명 배경"이 핵심 가치이며, 서식/테마는 가독성 보조 수단. 과도한 서식 옵션보다 기본값이 잘 동작하는 것이 중요
