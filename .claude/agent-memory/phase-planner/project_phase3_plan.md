Phase 3 계획이 2026-04-17에 수립되었다. MVP 완성 Phase로 Sprint 1(트레이+서식)과 Sprint 2(테마+메뉴 완성)로 구성.

**Why:** Phase 1-2 기반 위에 F4(서식), F5(테마), F9(트레이), F11(메뉴)을 구현하여 일상 사용 가능한 MVP 달성.

**How to apply:**
- 투명도 조절은 슬라이더 대신 5단계 선택 (20/40/60/80/100%)으로 확정
- 글꼴/크기/색상은 프리셋 + 다이얼로그 조합
- 컨텍스트 메뉴 서브메뉴 최대 1단계
- NotifyIcon Dispose 처리 필수 (정상/비정상 종료 모두)
- Click-Through는 Phase 4이므로 트레이 메뉴에 포함하지 않음
