# OverlayNotepad

화면 위에 항상 떠 있는 투명 메모장.
배경을 투명하게 하여 뒤의 콘텐츠를 보면서 메모할 수 있는 Windows 오버레이 도구입니다.

```
┌─────────────────────────────────────┐
│  [드래그 영역 - 상단 바]             │  ← 드래그로 이동
├─────────────────────────────────────┤
│                                     │
│   투명 배경 위에 텍스트 표시         │
│   (테두리+그림자 효과 적용)          │
│                                     │
│   여기에 메모를 입력합니다...        │
│                                     │
└─────────────────────────────────────┘
```

---

## 왜 만들었나

듀얼 모니터가 없는 환경에서 API 문서를 보면서 코딩하거나, 강의 화면 위에 필기하거나, 원문을 보면서 번역 메모를 남기고 싶을 때 — 화면 위에 투명하게 떠 있는 메모장이 필요합니다.

기존 메모장은 배경이 불투명해서 뒤의 콘텐츠를 가리고, 스티키 노트는 투명도 조절이 안 됩니다. OverlayNotepad는 배경을 완전히 투명하게 만들어 뒤의 화면을 그대로 보면서 메모할 수 있습니다.

---

## 주요 기능

### 투명 배경

두 가지 투명 모드를 지원합니다.

- **배경 투명 모드** — 배경만 투명, 텍스트는 불투명 유지. 뒤의 콘텐츠를 보면서 선명한 글자로 메모
- **전체 투명 모드** — 배경과 텍스트 모두 투명도 조절. 읽기 전용 참고 메모에 적합

투명도는 우클릭 메뉴에서 단계별로 조절할 수 있습니다.

### 텍스트 테두리와 그림자

투명 배경 위에서 어떤 색상의 콘텐츠가 뒤에 있든 글자가 잘 보여야 합니다.
글자에 테두리(아웃라인)와 그림자 효과를 적용하여 밝은 배경이든 어두운 배경이든 가독성을 확보합니다.
테두리/그림자 색상을 직접 선택하거나 효과를 끌 수 있습니다.

### Click-Through

Click-Through 모드를 켜면 마우스 클릭이 메모장을 통과하여 뒤의 앱에 전달됩니다.
메모를 화면에 띄워두고 뒤의 앱을 자유롭게 조작할 수 있습니다.
글로벌 핫키(`Ctrl+Shift+T`)나 시스템 트레이 메뉴로 전환합니다.

### 그 외

- **Always on Top** — 항상 최상위 표시 토글
- **서식** — 글꼴, 크기, 색상 변경 (전체 텍스트 일괄 적용)
- **다크/라이트 테마** — 원클릭 테마 전환
- **자동 저장** — 텍스트 변경 후 2초 뒤 자동 저장, 비정상 종료 시에도 데이터 보존
- **윈도우 관리** — 타이틀바 없는 깔끔한 외관, 자유 리사이즈, 위치/크기 기억
- **시스템 트레이** — 최소화 시 트레이 숨김, 더블클릭 복원
- **글로벌 핫키** — `Ctrl+Shift+N`으로 다른 앱에서도 표시/숨김 토글
- **컨텍스트 메뉴** — 우클릭으로 모든 설정에 빠르게 접근

## 설정

모든 설정은 `%AppData%/OverlayNotepad/settings.json`에 자동 저장됩니다.

| 항목 | 기본값 |
|------|--------|
| 윈도우 크기 | 400 x 300 |
| 투명도 | 80% |
| 투명 모드 | 배경 투명 |
| Always on Top | 켜짐 |
| 테마 | 다크 |
| 글꼴 | 맑은 고딕, 14pt |
| 텍스트 테두리 | 켜짐 |

## 기술 스택

| 항목 | 선택 |
|------|------|
| 프레임워크 | .NET Framework 4.8 |
| UI | WPF (XAML + 코드비하인드) |
| 투명 윈도우 | WPF AllowsTransparency + WindowStyle=None |
| 텍스트 효과 | FormattedText + Geometry (아웃라인), DropShadowEffect (그림자) |
| Click-Through | Win32 WS_EX_TRANSPARENT 인터롭 |
| 글로벌 핫키 | Win32 RegisterHotKey 인터롭 |
| 시스템 트레이 | WinForms NotifyIcon 인터롭 |
| 설정 저장 | JSON (수동 파싱, NuGet 의존성 없음) |

**외부 라이브러리 없음.** .NET Framework 기본 제공 라이브러리만 사용하여 단일 EXE로 배포합니다.

## 시스템 요구사항

- Windows 10 (1903+) 또는 Windows 11
- .NET Framework 4.8 (Windows 10 이상에 기본 포함)
- 설치 불필요 — EXE 파일 하나만 실행
